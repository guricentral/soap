1. 비누제조기가 뭔가요?

환자와 대화한 내용(진료 메모, 녹음, 녹취록 등)을 넣으면 AI가 자동으로 SOAP 노트로 변환해주는 웹앱입니다.

✅ 여러 AI를 동시에 돌려서 결과 비교
✅ 📋 EMR 복사 버튼 → EMR에 바로 붙여넣기
✅ 선생님의 진료 스타일·약제·구비약물 반영
✅ 진단명 기반 의료 교육 이미지 자동 생성
✅ MRI/CT 영상 첨부 → AI가 분석하여 Assessment 반영

2. 시작하기 (5분이면 끝!)

① 접속

https://guricentral.github.io/soap/
Chrome, Edge, Safari 모두 가능. 즐겨찾기 해두세요!

② API 키 입력 (처음 1회만)

API 키 = AI 서비스 비밀번호. 전부 입력할 필요 없습니다! 아래 3개면 무료로 전부 됩니다.

키발급처가격용도
Groqconsole.groq.com무료⭐ 음성인식(STT) + 텍스트 AI
Cerebrascloud.cerebras.ai무료초고속 텍스트 AI
Geminiaistudio.google.com무료텍스트 AI + 이미지 생성
API 키 발급 방법 (Groq 예시)
  1. console.groq.com 접속
  2. 구글 계정으로 로그인
  3. API Keys 메뉴 → Create API Key
  4. 생성된 키(gsk_...)를 복사
  5. 비누제조기의 Groq 입력란에 붙여넣기 → 끝!

③ 진료과 선택

드롭다운에서 본인 과 선택 → 기본 설정 자동 적용 → 💾 저장 클릭

④ SOAP 변환!

  1. AI 모델 체크 (🏆Best 버튼 추천)
  2. 진료 내용 입력 (타이핑/녹음/파일)
  3. 🧼 SOAP 변환 시작! 클릭
  4. 결과 비교 → 📋 EMR 복사

4. 음성인식(STT) 엔진 선택 가이드

v1.4부터 5종의 음성인식 엔진을 선택할 수 있습니다. 녹음 탭/음성파일 탭의 드롭다운에서 변경하세요.

상황추천 엔진
일상 진료 (가성비 최고)🚀 Groq Turbo ⭐
정밀한 녹음🚀 Groq V3

💡 화자분리: GPT-4o Transcribe를 선택하면 "의사: …" "환자: …" 이렇게 구분됩니다. OpenAI 키 필요 (유료).

5. 입력 방법별 안내

📋 타이핑/붙여넣기 — 가장 간단

진료 메모를 자유롭게 타이핑하거나, 클로바노트 녹취록을 Ctrl+V로 붙여넣기. 형식 상관 없습니다.

50대 남자 허리 통증 우측 다리 저림 3주
SLR 양성 VAS 7
MRI 안 찍음 고혈압약 복용 중
MRI 찍고 가바펜틴 시작 예정

🎙 마이크 녹음

외장 마이크(Jabra 등)로 직접 녹음 → STT 자동 변환. 녹음 버튼 탭 → 진료 → 다시 탭으로 중지.

🎵 음성파일 업로드

핸드폰, 클로바노트 등 녹음 파일을 업로드. m4a, mp3, wav, webm 지원 (25MB 이하).

📄 문서 업로드

txt, md 파일 → 내용이 자동으로 입력란에 채워집니다.

6. 진료과별 특수 기능

진료과 선택 시 세부전공, 환자군, 약제, 진료철학이 자동으로 채워집니다. 자유롭게 수정 가능!

진료과SOAP 모드
🧠 신경외과 · 🦴 정형외과 · 🔪 일반외과 · 💊 내과 · 😴 마취과표준 SOAP
🩻 영상의학과판독소견 모드

🩻 영상의학과 판독 모드:

S → Clinical Indication (의뢰 사유)
O → Findings (판독소견) — 가장 상세하게 작성됨
A → Impression (핵심 진단)
P → Recommendation (추가 검사/f/u 권고)

8. 자주 묻는 질문

Q: 정말 무료인가요?

네. Groq, Cerebras는 완전 무료입니다 (신용카드 불필요). 하루 30건 진료 충분합니다.

Q: 환자 데이터가 저장되나요?

아닙니다. AI API로 전송하여 SOAP 생성 후 어디에도 저장되지 않습니다. 브라우저를 닫으면 사라집니다.

Q: Groq Turbo vs OpenAI Whisper?

같은 Whisper 모델이라 정확도 비슷합니다. Groq가 9배 싸고 228배 빠릅니다. Groq 추천!

Q: 화자분리가 뭔가요?

GPT-4o Transcribe 선택 시 "의사: …" "환자: …" 이렇게 누가 말했는지 구분해줍니다. OpenAI 키 필요(유료).

Q: 다른 PC에서도 쓸 수 있나요?

네. 웹 주소만 접속하면 됩니다. API 키와 프로필은 브라우저별 저장이라 새 PC에서 다시 입력 필요.

Q: 녹음 버튼이 안 눌려요

HTTPS 환경이 필요합니다. GitHub Pages 주소(https://guricentral.github.io/soap/)로 접속하면 자동 해결됩니다.

Q: SOAP 결과가 마음에 안 들면?

상세도(간략/보통/상세), 언어(영어위주/한영병기) 변경하거나, 개인 커스텀 규칙 추가, 또는 다른 AI 모델 시도해보세요. 모델마다 스타일이 다릅니다.
